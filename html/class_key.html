Detailed Description

template<gpio_num_t Key_pin>
class Key< Key_pin >

Class that represents one key of device keyboard.

This is a Meyers's singleton template; parameterized by elements of gpio_num_t type: Key_pin which is a esp32 pin, key is hardwired to.
